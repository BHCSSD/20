You will create a program that plays the "FizzBuzz" game up to a **user-specified limit**. Reminder to make this professional, separate your UI from your **Logic**.

> [!IMPORTANT]
> **The Golden Rule for this Assignment:**
> The `main()` function is the only place where `input()` is allowed.
> Your helper functions (`forFizz` and `whileFizz`) should only receive data through **parameters** and handle the **output**.
> You are **not** allowed to use global variables

## **Requirements**

### **1.`main()`**
Your `main()` function will handle all user interaction. 
* **Input Validation:** Ask the user for a limit. Use a `while` loop and `.isdigit()` to keep asking until they provide a valid positive integer.
   * **NOTE TO FUTURE ME:** use `isinstance()`instead of `isdigit()`
* **Customization:** Ask the user if they want to use the standard "Fizz" and "Buzz" or if they want to enter their own custom words
* **Mode Selection:** Ask the user if they want to run the **(F)or** version or the **(W)hile** version.
* **The Call:** Pass the limit and the custom words into your chosen function as **arguments**.
