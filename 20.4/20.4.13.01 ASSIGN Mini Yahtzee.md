Build a simplified version of Yahtzee where the computer automatically helps the user make the best tactical decision. 

### **The Game Rules:**
1. **The First Roll:** Roll three dice.
2. **The Strategy:** * **Three of a Kind:** You hit the jackpot! Keep all three.
   * **A Pair:** Keep the matching pair, re-roll the third "kicker" die.
   * **No Matches:** Keep only the highest die and re-roll the other two.
3. **The Scoring:** Calculate the final score based on the results of the second roll.

---

## **Requirements**

### **1. `rollDice()`**
* **Goal:** Return a single random integer between 1 and 6.
* **Higher level:** Make another helper funciton as visualizer or a dictionary that maps the number to a Unicode symbol (⚀ ⚁ ⚂ ⚃ ⚄ ⚅) to make the console output look professional.

### **2. `calculateScore(d1, d2, d3)`**
* **Logic Only:** This function should **not** contain any `print()` or `input()` statements. It only takes three numbers and returns one number.
* **Scoring Math:**
    * **Three of a Kind:** 20 + (d1 + d2 + d3)
    * **A Pair:** 10 + (the sum of the two matching dice)
    * **High Die:** The value of the highest die (aka, `3-5-2` score = 5).

### **3. `get_reroll_choice(d1, d2, d3)`**
* **The Strategy Engine:** Write a function that takes the first three rolls and returns a "plan." 
* It should return which dice need to be rolled again based on the rules in the Objective. 
* *Hint: Using `max()` and `if/elif` statements will be your friend here.*

### **4. `main()`**
* As always, `main()` handles the storytelling:
    * Call `rollDice()` three times and show the user their "Hand."
    * Call `get_reroll_choice()` to decide the strategy.
    * Print a message explaining the strategy (e.g., *"Found a pair of 5s! Re-rolling the 2..."*).
    * Show the final hand and call `calculateScore()` to show the result.

### **5. `testScore()`**
* You must verify your math. Use `assert` to test all three scoring tiers:
  * 3 of a kind
  * a pair
  * no matching
