> [!NOTE]
> **How to Write Correct Python Asserts**
> In Python, `assert` is a keyword, not a function. You evaluate an expression to see if it results in `True`. If it does, nothing happens, and your program keeps running. If it evaluates to `False`, your program instantly crashes with an `AssertionError`.
> * **Correct Syntax (No Outer Parentheses):** `assert balance == 100`
> * **Why parentheses hurt:** Writing `assert(x == y)` works fine for simple checks, but if you accidentally type a typo like `assert (x == y, "Error")`, Python interprets this as a non-empty tuple. In Python, non-empty tuples always evaluate to `True`, meaning your test will **pass even if your code is completely broken**! Stick to standard keyword spacing.

## Part 4: Architecture Challenge

### Drill 4.1: The Ticket Booth Program

Create a structured Python program containing exactly three functions: `calculate_ticket_price`, `get_clean_tickets`, and `main`.

Follow the architectural design constraints below precisely:

1. **`calculate_ticket_price(ticket_count)`**
* **Role:** Pure Computational Math.
* Each ticket costs `$12`. If a user buys **5 or more** tickets, they get a flat `$10` discount off the total price.
* *Constraint:* No `input()` or `print()` inside this function. It takes an integer and returns a numeric price.
* **Write `assert` statements for this function:** Protect this math! Write one assertion checking the price of `3` tickets ($3 \times 12 = 36$) and another checking the price of `5` tickets ($5 \times 12 - 10 = 50$).


2. **`get_clean_tickets()`**
* **Role:** Input Validation.
* Loops indefinitely until the user types a positive string of digits via `input()`.
* *Constraint:* Must use `.isdigit()`. Returns the safe converted integer.


3. **`main()`**
* **Role:** Controller.
* Calls `get_clean_tickets()` to safely acquire the amount from the user.
* Passes that amount into `calculate_ticket_price()`.
* Prints the final bill to the user using a clean f-string format showing two decimal points (e.g., `Total: $50.00`).
