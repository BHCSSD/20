## **Quick Reference Guide for this assignment**

### **1. Boolean Logic Rules**

Python uses plain English words instead of JavaScript symbols.

* Use `and` instead of `&&`
* Use `or` instead of `||`
* Use `not` instead of `!`
* **Capitalization matters:** Always use `True` and `False`

### **2. f-Strings (Template Literals)**

Prefix your text quotes with `f` and use curly braces `{}` to inject variables or simple expressions.

```python
score = 100
print(f"Your final score is: {score}")

```

### **3. Loops and the range() Function**

Python loops depend entirely on colons (`:`) and a 4-space indent block.

* `range(stop)` counts from 0 up to, but not including, `stop`.
* `range(start, stop)` counts from `start` up to, but not including, `stop`.
* `range(start, stop, step)` changes the increment index. Use a negative step (e.g., `-1`) to count backward.
