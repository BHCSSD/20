You will create a program that prints the lyrics to the song **"99 Bottles of [Drink] on the Wall."** 
You’ll build two versions of the song logic and a main function to handle the user interface.

---

## **Requirements**

### **1. The Brain: `main()` Function (30%)**
Your `main()` function will act as the controller for the whole program. It must:
* **Ask for Age:** Prompt the user for their age. 
    * *upper level:* Use `.isdigit()` to check if the input is a number so the program doesn't crash if they type "twenty."
* **Determine the Drink:**
    * If your user is over 18 ask if they prefer `beer` or `pop`.
    * If your user is under 18, automatically set the drink to `pop`.
* **Choose the Version:** Ask the user if they want to run the **(F)or loop** version or the **(W)hile loop** version.
* **Execution:** Based on their choice, call either `forLyrics()` or `whileLyrics()` and pass the drink and/or bottle count as **arguments**.

  
* **this is the only function where you are going to have user input.**

---

### **2. Version 1: `forLyrics()` (30%)**
Write a function that accepts the drink type as a parameter and uses a **`for` loop** to count down from 99 to 0.
* **Requirement:** Use `range()` to count backward. 
* **Comment Task:** Include a comment explaining what the three numbers inside your `range()` function do
* **F-Strings:** Use f-strings for all lyric lines.

Reminder: what parameters will you need to make this?  

---

### **3. Version 2: `whileLyrics()` (30%)**
Write a function that uses a **`while` loop** to print the lyrics.
* **Custom Start:** Sometime you don't want 99 bottles, add in a way to ask your user
* **Counter Logic:** You must manually decrease the bottle count inside the loop.
* **User Input:** In `main()`, if the user chooses the while version, ask them: *"How many bottles should we start with?"*
Reminder: what parameters will you need to make this?  
---

### **4. Grammar & Logic (10%)**
Your song shouldn't look "robotic." Ensure your loops handle these cases:
* **Plurality:** If there is **1 bottle**, it should say "1 bottle," NOT "1 bottles."
* **The End:** When the count hits **0**, it should print: 
  > *"No more bottles of [drink] on the wall!"*
