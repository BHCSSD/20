## Step 1: Defining the Map Layout
* Instead of defining everything in raw pixels, we use a **grid layout** object.
* Each feature type contains an array of `[col, row]` coordinate pairs.

```javascript
let layout = {
  trees: [[2, 2], [2, 3], [5, 8], [6, 8]],
  mountains: [[0, 0], [1, 0], [0, 1]],
  roads: [[4, 0], [4, 1], [4, 2], [4, 3], [5, 3], [6, 3]]
};

let cols = 10
let rows = 10
let cellSize  //placeholder

```

---

## Step 2: Setting up the Canvas

* We calculate `cellSize` dynamically by dividing the canvas width by the total number of columns. This guarantees everything scales perfectly.

```javascript
function setup() {
  createCanvas(400, 400);
  cellSize = width / cols; // 400/10=40 pixels per grid block
  textAlign(CENTER, CENTER);
}

function draw() {
  background(100, 200, 100); // Grass green background
  drawGridLines();

  // Step 5: Draw each category from the object layout
  drawFeature(layout.roads, color(150, 150, 150), "🛣️");
  drawFeature(layout.trees, color(34, 139, 34), "🌲");
  drawFeature(layout.mountains, color(112, 128, 144), "⛰️");
}

```

## Step 3: The `drawFeature()` Helper

* This single function handles rendering any environment type by looping through its grid array, converting grid spaces to pixels, and stamping down a color background and an emoji overlay.

```javascript
function drawFeature(arr, col, emoji) {
  // We loop through the array of pairs
  for (let i=0; i<arr.length; i++) {
    //Step 1: Access each X, Y pair (col, row)
    let gridX = arr[i][0]
    let gridY = arr[i][1]

    //Step 2: Scale it up to our map pixel size
    let pixelX = gridX * cellSize
    let pixelY = gridY * cellSize
    
    //Step 3: Draw the block background outline
    fill(col)
    noStroke()
    rect(pixelX, pixelY, cellSize, cellSize)
    
    //Step 4: Input emoji/pictures centered in the cell
    fill(255)
    textSize(cellSize * 0.6) //change size text to fit the cell
    text(emoji, pixelX + cellSize/2, pixelY + cellSize/2)
  }
}

```

---

## Step 4: Drawing Helper Grid Lines

```javascript
function drawGridLines() {
  stroke(0, 40); // Soft alpha transparency so lines aren't too harsh
  for (let i = 0; i <= cols; i++) {
    line(i * cellSize, 0, i*cellSize, height)
    line(0, i * cellSize, width, i*cellSize)
  }
}

```

> [!TIP]
> Separating map data (the grid layout) from map rendering (the `drawFeature` function) makes it incredibly easy to change your level design later without modifying your drawing logic.
