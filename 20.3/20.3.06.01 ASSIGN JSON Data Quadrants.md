# 20.3.06.01 ASSIGN JSON Data Quadrants.md

**TODO:**
Build a Game Status Interface for a fantasy RPG. Instead of hardcoding player stats, you will load all game information from four separate JSON files. 

---

1. Make an assessts folder
2. indside create the four JSON files below
3. follow the specific drawing instructions for each.

#### **Quadrant 1: The Player stat block `hero.json`**
* **Goal:** Make a stat block that displays all properties from a basic object.
* **JSON:** Start with this and **add 3 more properties** of your choice for each character
    ```json
    {
      "heroName": "Aeliana",
      "type": "Mage",
      "mana": 180,
      "primaryColour": [138, 43, 226]
    }
    ```
* **The Task:** Display the `heroName` and `job` as text. Use `mana` to set the width of a rectangle (Mana Bar) and `primaryColor` as the fill.

#### **Quadrant 2: Inventory `items.json`**
* **Goal:** Use a loop to display a list of values from the main character's inventory.
* **JSON:** You must have **at least 7 items** in your inventory array.
    ```json
    {
      "inventory": [.....]
    }
    ```
* **The Task:** Loop through the `inventory` array as a vertical list

#### **Quadrant 3: Party Members `party.json`**
* **Goal:** Loop through an array of objects to display individual stats.
* **JSON:** Your party must have **at least 4 players**.
    ```json
    {
      "members": [
        { "name": "Korg", "hp": 120, "role": "Tank" },
        { "name": "Zia", "hp": 85, "role": "Healer" }
      ]
    }
    ```
* **The Task:** Loop through the `members` array. Display each member's `name` and `role`. Use the `hp` value to determine the size or color intensity of a status icon for each person.

#### **Quadrant 4: Mini Map `world.json`**
* **Goal:** Use nested arrays for roads and elements and use images for the display
* **JSON:** ```json
    {
      "locations": [
        { "type": "castle", "coords": [40, 50] },
        { "type": "forest", "coords": [180, 200] }
      ],
      "roads": [
        { "name": "King's Way", "path": [[40, 50], [100, 100], [180, 200]], "color": [150, 75, 0] },
        { "name": "East Trail", "path": [[180, 200], [250, 250]], "color": [100, 100, 100] }
      ]
    }
    ```
* **The Task:** * **Roads:** Loop through `roads`. Access the nested `path` to draw lines connecting the points.
    * **Locations:** Loop through `locations`. Load your images in `sketch.js` and use the `coords` to place them.
    * **Visuals:** Use an `if/else` or `switch` based on `type` to choose different shapes or images.

> [!tip]
>
> For the roads notice you have nested arrays.
> Think of it like a **row** and a **column**. 
> `let path = [[40, 50], [100, 100], [180, 200]]`
>  
> 1.  **First Bracket `[i]`**: Selects the "package" (the inner array).
> 2.  **Second Bracket `[j]`**: Opens the package to get the number.
>  
> ### **Cheat Sheet**
> * **`path[0]`** gives you `[40, 50]` (The whole first x,y point)
> * **`path[0][0]`** gives you  `40` (The **X** of the first point)
> * **`path[0][1]`** gives you  `50` (The **Y** of the first point)


## Starter
```js
function setup() {
 createCanvas(800, 800)
 textAlign(CENTER, CENTER)
 textSize(22)
 }


function draw() {
 background("purple")
   drawQuadrantDividers()
   labelQuadrants()


}


// Helper function to draw quadrant lines and labels
// no need to change
function mousePressed(){
 print("x:",mouseX,"y:",mouseY)
}


function drawQuadrantDividers() {
 stroke(100);
 line(width/2, 0, width/2, height);
 line(0, height/2, width, height/2);


}


function labelQuadrants() {
 noStroke();
 fill(0);
 textSize(16);
 textAlign(CENTER, CENTER);
 text('Q1', width/4, 20);
 text('Q2', (3*width)/4, 20);
 text('Q3', width/4, (height/2)+20);
 text('Q4', (3*width)/ 4, (height/2)+20);


}


```
