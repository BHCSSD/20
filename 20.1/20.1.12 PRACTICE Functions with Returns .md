### 5. Multiple Returns

```javascript
function testNum(x) {
 let newX = (x*2)


 if ( (newX/3) > 10) {
   return "Big";
 } else {
   return "Small";
 }
}
console.log(testNum(15.01));
console.log(testNum(12));
```
<details><summary>Answer</summary>
  
   "Small"
   "Big"
   Make sure you have the "" in the response
 </details>
