### **1. If we call `updateScore(5, 3)`, what value is returned?**

```javascript
function updateScore(currentScore, points) {
  return currentScore + points;
}

```

<details><summary>Answer</summary>

**8** (The function takes the two inputs and returns their sum).
</details>

### **2. After this code runs, what is the value of `result`?**

```javascript
function doubleValue(n) {
  let temp = n * 2;
  return temp;
}

let result = doubleValue(10);

```

<details><summary>Answer</summary>

**20**. The local variable `temp` is used to calculate the value, which is then sent back to the `result` variable.
</details>

---

### **3. Does calling this function change the global `total`?**

```javascript
let total = 10;

function calculate(total) {
  total += 5;
  return total;
}

calculate(total);

```

<details><summary>Answer</summary>

**No, it is still 10.** Because `total` is passed as a parameter, the function creates a local copy. Since the result wasn't saved (e.g., `total = calculate(total)`), the global variable remains unchanged.
</details>

### **4. What is the value of `finalX`?**

```javascript
function move(pos, dist) {
  return pos + dist;
}

let x = 100;
let finalX = move(x, 50);
finalX = move(finalX, 50);

```

<details><summary>Answer</summary>

**200**. The function was used twice to increment the value by 50 each time.
</details>

### **5. If `checkLimit(15)` is called, what prints to the console?**

```javascript
function checkLimit(val) {
  let limit = 10;

  if (val > limit) {
    print("Over");
  } else {
    print("Under");
  }//end of if
}

```

<details><summary>Answer</summary>

**"Over"**. The parameter `val` (15) is compared against the local variable `limit` (10).
</details>

### **6. What is the value of `current` after these lines run?**

```javascript
function addBonus(score) {
  let bonus = 3;
  return score + bonus;
}

let current = 2;
current = addBonus(current);
current = addBonus(current);

```

<details><summary>Answer</summary>

**8**. The value starts at 2, becomes 5 after the first call, and 8 after the second.
</details>

### **7. What happens when this code runs?**

```javascript
function setup() {
  let speed = 5;
  let fastSpeed = boost(speed);
  print(boostAmount);
}

function boost(n) {
  let boostAmount = 10;
  return n + boostAmount;
}

```

<details><summary>Answer</summary>

It will **Error**. While `boost(speed)` correctly returns 15, the `print(boostAmount)` statement fails because `boostAmount` is **local** to the `boost` function and cannot be seen by `setup`.
</details>
